Windows 10是一個由微軟開發的作業系統，是Windows家族的成員，為Windows 8.1和Windows Phone 8.1的後繼者，開發代號為Threshold和Redstone（包含TH1和TH2、RS1和RS2），正式版已於2015年7月29日登場並開放符合條件的使用者免費升級。
Windows 10的設計目標是統一包括個人電腦、平板電腦、智慧型手機、嵌入式系統、Xbox One及Surface和Microsoft HoloLens等等，整個Windows產品系列的作業系統。它們共享一個通用的應用程式架構和Windows Store的生態系統。

Windows 10在測試階段發布個人電腦版的技术预览版、会员预览版（Insider Preview）和手機版的Mobile 移动预览版。
个人电脑[编辑]
2014年7月21日，外媒網站Neowin以及Myce網站洩露出Build 9795。其中包括了當時稱為Windows 9的Windows 10系統中的新的開始菜單以及窗口化Modern應用。[6][7][8]
2014年9月13日，德國網站Computer Base和WinFuture洩露出Build 9834。包括了虛擬桌面界面、改變的任務欄、未完成的通知中心、Charm菜單替代、Metro化的部分圖標，以及傳統桌面視窗無邊框化。[9][10]
2014年9月20日，微軟發布第一個Windows 10的技術預覽版Build 9841。[11]
技术预览版[编辑]
2014年9月30日，微软在美国舊金山发布会中亮相新一代Windows操作系统，并声称正式版将于2015年6月18日发行。随后10月1日便为注册Windows Insider会员计划的用户放出第一个技术预览版Build 9841，并提供镜像下载。此版本支持5种语言，开始菜单回归，加入任务视图，主要为滑鼠用户优化。
10月21日，Build 9860通过Windows 更新推送给Windows Insider会员用户，没有官方镜像提供下载，此版本主要加入获取预览版频率快慢的设置和初始版的通知中心。
11月13日，Build 9879通过Windows 更新推送给快速版Windows Insider会员用户，并提供镜像下载，但多数用户安装此版本后出现硬盘丢失等比较严重的问题。11月25日，重新修复的Build 9879推送给慢速版Windows Insider会员用户，此版本对Build 9860做了一些优化和界面调整，是2014年最后一个公开技术预览版。[12]
2015年1月21日，微軟在召开"Windows 10, The Next Chapter"的主题发布会上演示了Build 9924。1月24日，Build 9926通过Windows 更新向所有Windows Insider用户推送，同时提供镜像下载，新的Build 9926包含正體中文的25種新語言介面，同時新增平板模式，Cortana語音助理（仅美国英语时可用）等新特性。[13]
2015年3月19日，Build 10041通过Windows 更新推送给快速版Windows Insider会员用户，此版本添加了中文Cortana语音助理，恢复透明的任务栏和开始菜单等新特性。3月25日，Build 10041推送给慢速版用户，慢速版加入了3个补丁，修复了一些问题，并提供镜像下载。
2015年3月31日，Build 10049通过Windows 更新推送给快速版Windows Insider会员用户，此版本终于添加了翘首以盼的Project Spartan，但该版本没有官方镜像下载。
2015年4月23日，Build 10061通过Windows 更新推送给快速版Windows Insider会员用户，主要增加了全新的郵件和日曆通用應用，採用全新的暗黑風格主题，平板模式升级，取消創建多个虛擬桌面數量限制，回收站以及任务預覽模式圖標更新等，该版本同样没有官方镜像下载。
公开洩漏版[编辑]
2014年7月21日，Build 9795洩漏，该版本是最早被洩漏截图的一个版本，發現類似Windows 7樣式以及動態磚結合的开始菜单以及Modern視窗化應用。
2014年9月8日，Build 9834洩漏，内置Remind Me应用，也包括隐藏的MtcUvc应用等，开始菜单内发现Spartan程序但只有快捷方式。
2014年12月，有两个非微软官方推送的洩漏版本，即Build 9888和Build 9901。其中Build 9888标志着Windows 10的内核将由NT 6.4变为NT 10.0（但其实Build 9884才是第一个内核为NT 10.0的Windows 10），而Build 9901加入了原始的Cortana语音助理（仅限美国英语版）。
2015年4月5日，Build 10051洩漏，为官方版Build 10049添加了一些新功能，还包括全新"邮件"与"日历"应用以及贯穿于整个系统中细微改进。
4月11日，Build 10056洩漏，此版本将“个性化”整合到新版设置当中，优化平板模式，正式启用新版任务栏日历窗口等新特性。
Insider Preview[编辑]
Insider Preview更加接近RTM。Insider Preview是一個頻道，在Windows 10 RTM發布後，仍會發布新組建以測試新功能和新介面。
2015年4月29日，在微软Build 2015开发者大会结束后不久，Build 10074通过Windows 更新推送给所有Windows Insider会员用户，同时提供镜像下载。 为感谢370万Windows Insider会员用户对Windows 10的不断改进反馈，Build 10074由Technical Preview（技术预览版）改为Insider Preview（會員预览版）。
Threshold[编辑]
2015年7月15日發佈Threshold 1的RTM前最後一個預覽版Build 10240，這也是在7月29號對全球公開發布的RTM版。[14]
2015年11月12日發佈Threshold 2(簡稱TH2)電腦版重大更新[15][16]，Windows 7/8.1的使用者可直接升級至此版本，至於在7月29日發布的Windows 10如果是從舊版本升級安裝，則需要等到安裝時間超過31天才能進行。[17]
Redstone[编辑]
在Threshold 2正式版發布之後，Windows Insider用戶已進入rs1_release通道，開發代號Redstone，而代號Redstone名稱來自Minecraft的紅石[18]，Redstone更新即將分別在2016年7月29日以及2017年的春季發布2次更新。[19][20]而Redstone 3預定在2017年下半年發布。[21]RS1的主要任務是將PC、Xbox One、手機等不同設備上的Windows 10真正統一起來，包括完善通用應用平臺、提供Xbox應用商店、支援Xbox/PC跨平臺遊戲、Continuum模式（可通過產品發訊息和打電話），以及各種媒體功能。而RS2會繼續完善RS1裡未完成的更新內容，並且還會繼續從各種細節上強化Windows 10，包括使用者介面、新功能，比如說開始功能表和設置應用的更新，動作中心和通知中心的增強，控制台的持續削弱等等。TH2系統版本號是1511，而最新曝光的RS1的系統版本號很可能會在1607。根據曝光的資訊來看，RS1的正式版可能會於2016年夏天發佈，而RS2則會在2016年年末或者是2017年的年初發佈。[22]

预览版[编辑]
2014年10月1日，微软开发布会宣布新一代操作系统Windows 10，次日微软提供Windows 10技术预览版下载。系统命名不是盛传已久的Windows TH、Windows X、Windows One，甚至不是Windows 9。在开发阶段曾泄露出新一代Windows内部代号为Windows Threshold。至於為何跳過9而是10，原因眾說紛紜：官方說法是因為Windows 10進步幅度大，因此改用10；但有媒體認為，跳過9是為了避開部分程式語言將「Windows 9」誤判為「Windows 95」或「Windows 98」而產生的錯誤。[28]




Windows 10开始功能表


微软旨在服务客户，称Windows 10为「有史以来最全面的操作系统（most comprehensive platform ever）」及「至今最好的Windows（The best Windows ever）」。Windows 10恢复了类似Windows Vista、Windows 7的传统开始按钮，但是在细节上糅合Windows 8和Windows 8.1的微软“Modern瓷砖”设计。Windows 10作为微软下一代系统的统一品牌名称，将覆盖所有尺寸和品类的Windows设备，可无缝运行于微型计算机（类似英特尔伽利略）、手机（ARM芯片）、平板（ARM和x86芯片）、二合一设备、桌面电脑及服务器等几乎所有硬件。Windows 10贯彻了“移动为先，雲为先”（mobile first, cloud first）的设计思路，三屏一云，多个平台共用一个应用商店，应用统一更新和购买，是跨平台最广的操作系统。
正式版[编辑]
2014年4月，微軟作業系統事業群執行副總裁Terry Myerson在Build開發者大會的演講上宣佈，將針對8寸（包括8寸）螢幕以下的裝置（包括手機及小型平板，以及物聯網裝置），提供免費的Windows作業系統。[29]
2015年1月21日，微軟宣佈，Windows 7 SP1、Windows 8.1與Windows Phone 8.1的用戶可在第一年內免費升級至Windows 10。[30]但是，10天後，就立刻修正為使用企業版Windows的用戶還是得另外付費才能升級。[31]針對Raspberry Pi 2的研發者，微軟將免費提供Windows 10。[32]
中國大陸的Windows用戶可以透過奇虎360和騰訊下载Windows 10升级[30]；至於盜版及錯誤授權的Windows，雖然也可免費升級，但不會因此而變成正版和享有技術支援。[33]
2015年6月1日，微軟正式宣布於7月29日推出Windows 10[34]，並隨Windows Update提醒符合升級條件的用戶如何預約取得Windows 10。完成預約升級的用戶，會在「取得Windows 10」應用程式與Windows Update看到相關通知。
自2015年7月29日起，分批陸續發行正式版更新。首日即有1400萬裝置更新成功。[35] 截至2015年8月27日，已有7500萬次下載。[36] 截至2015年10月9日，已有1.1億次下載。[37]
與先前的Windows一樣，微軟仍提供實體盒裝版本在市面上販售，但因近幾年越來越多電腦沒有配備光碟機，實體盒裝版本不再提供安裝光碟，僅提供支援USB 3.0可開機安裝的USB隨身碟。
特色[编辑]
[38]Windows 10采用全新的開始功能表，微软動態磚界面仍存續，并且可以根据用户的偏好快速切換。
Windows 10大幅修改了系統的用戶介面，當中包括加入由Windows 7及Windows 8.1改良的混合「開始」功能表，以取代Windows 8.1的「開始」畫面。新功能表融合了Windows 8「動態磚」、虛擬桌面設計以及通知中心側欄（取代舊版本的charms bar）。Windows 10提供更多微軟服務整合，例如個人助理Microsoft Cortana及Xbox Live，提供了新的瀏覽器Microsoft Edge、Windows Hello（僅適用於部分裝置）以及新版Direct X及WDDM來改善遊戲圖形方面的性能[39]。

开始功能表和通知中心：熟悉的桌面开始菜单在Windows 10正式归位，它融合了Metro风格的界面，传统与现代两者结合在一起。開始功能表可自由切换全螢幕或一般大小。由螢幕右下角觸發的通知中心，可快速調整設定和一覽通知。


虚拟桌面和工作檢視：微软从OS X和Linux等，借鉴“Multiple desktops”功能。该功能可让用户在同个系统下使用複數个桌面，用户可以根据自己的需要，在不同桌面环境间进行切换。微软Windows在该功能上加入在「工作檢視」模式，並增加程式排列，程式会以某种推荐的排版显示在桌面环境中，点击右下侧的加号即可添加一个新的虚拟桌面。


視窗化程式：現代化应用程式现在以一般桌面程序的方式，視窗化地运行：可以随意拖曳、調整大小，也可以最小化、最大化、全螢幕以及关闭程式。Windows 10的螢幕切割功能增强，现在可在屏幕中同时平均設置四个視窗，Windows 10还会在单独窗口内显示正在运行的其他应用程序。微软在Windows 10侧边新加入一个“Snap Assist”按钮，通过它可以将多个不同桌面的应用程式展示在此，并和其他程式自由组合成多任务模式。


多工管理界面：界面类似苹果OS X系统，任务栏中出现一个全新的按键“任务查看（Task View）”。桌面模式下可以运行多个应用和对话框，并且可以在不同桌面间自由切换。这个功能类似苹果OS X的Exposé，能将所有已开启窗口缩放并排列，以方便用户迅速找到目标任务。通过点击该按钮可以迅速的预览多个桌面中打开的所有应用，点击其中一个可以快速跳转到该页面。传统应用和桌面化的Metro应用在多任务中可以更紧密的结合在一起。


CMD：Windows 10在指令提示符（Command Prompt）中增加了对粘贴键（Ctrl+V）的支持。






上圖為Windows 10的任务栏，图中由左而右分别为Cortana、工作檢視、Microsoft Edge、文件资源管理器和Windows Store。



Continuum：對於變形平板等觸控裝置，或是不想使用鍵盤與滑鼠時，平板電腦模式可讓 Windows 的使用更為容易且更加直覺。即使是無觸控的電腦，也會有此模式，此模式會改變工具列、開始功能表、通知中心的細節，並有單獨視窗排列。


Windows Hello：Windows Hello 是一個更個人化的 Windows 10 裝置登入方式，只要看一下或觸碰一下即可登入，不需要輸入任何密碼。大部分具有指紋辨識器的電腦現在都已經可以使用 Windows Hello，而且也即將出現更多能夠辨識臉部與虹膜的裝置。


Windows Store：Windows Store市集是可一次購足音樂、影片、遊戲與應用程式的集中式商店。可於購買之前先試用應用程式，或挑選免費的應用程式。 Windows 10 應用程式可在所有的 Windows 10 裝置上運作。


Microsoft Edge：Microsoft Edge是第一個可直接在網頁上製作筆記、書寫、隨手塗鴉，以及醒目提示的瀏覽器。使用閱讀清單圖示儲存最愛的文章以供稍後閱讀，然後再閱讀檢視它們。能輸入的地方就能寫，Microsoft Edge 不是可以在其中書寫的唯一 app。 使用平板電腦手寫筆、手指、或滑鼠，可以輸入的地方都能書寫，或只是在 OneNote 中塗鴨。


相片：全都集中在一處。相片app可從手機、電腦和OneDrive將所有的相片和影片收集於一處，然後它會將回憶整理成相簿。


相容性：Windows 10重視不同裝置之间的协调，尤其是在运行Windows版本的电脑和智慧型手机设备。Windows 10 電腦的Windows Runtime应用程式可以移植到Windows 10 Mobile，并分享代码库。Windows 10 Mobile與个人电脑版本分享同樣的用户界面元素。[40][41]


Windows Ink:具有多重書寫工具等功能。


Microsoft Cortana:語音助理，可查詢資料或建立行事曆等。

裝置与平台统一[编辑]
Windows 10恢复原有的開始功能表，并将Windows 8系统中的“开始界面”整合至其中。現代化應用程式则在桌面以視窗化模式运行。[42]
Windows 10将为所有硬體提供一个统一的平台。Windows 10将支持广泛的设备类型，从物联网设备到全球企业数据中心服务器。其中一些设备屏幕只有4英寸，一些设备的屏幕则有80英寸，有些甚至没有屏幕。有些设备是手持类型，有的则需要远程操控。这些设备的操作方法各不相同，手触控、笔触控、鼠标键盘，以及动作控制器，微软都将全部支持。这些设备将会拥有类似的功能，微软正在从小功能到云端整体构建这一统一平台，跨平台共享的通用技术也在开发中。
Windows 10覆盖所有尺寸和品类的Windows设备，所有设备将共享一个应用商店。因为启用了Windows Runtime，用户可以跨平台的在Windows设备（手机、平板电脑、个人电脑以及Xbox等）上运行同一个应用。[43][42]
Windows 10 Mobile[编辑]
主条目：Windows 10 Mobile
参见：Microsoft Lumia 950 # 作業系統
Windows 10 Mobile，中文译為Windows 10移动版，是Windows 10作業系統的分支版本。此版本是专为屏幕尺寸低于8寸的智能手机和平板电脑运行。它採用ARM架构和IA-32处理器架构。該版本為移动设备提供了强大的功能。其中包括：同步處理功能、通用应用&#160;（英文：Universal Apps）、从Android和iOS的平台移植的应用、将裝置连接至外部显示器、可使用类似PC的滑鼠和键盘输入。Windows Phone 8.1智能手机将可升级Windows 10 Mobile，[44]但一些功能取决于硬體設備的兼容性。[45][46]微软已经保证未来将添加更多非Lumia机型的支持。 [47]Windows 10 Mobile目前微軟表示正式版大概在12月陸續發布。[48][49]
Windows 10重視不同设备之间的同步，尤其是在设备运行Windows版本的电脑和智慧型手机。Windows 10電腦的Windows Runtime应用程式可以移植到Windows 10 Mobile，并分享共同的代码库。Windows 10 Mobile與个人电脑版本分享同樣的用户界面元素，但与桌面版本不同的是，不会有多用户登录功能。[40][50]
Windows 10 Mobile的手機若接到Windows 10電腦（甚至是無主機的螢幕），則會可以在開始功能表或全螢幕顯示整合化的手機畫面。
安全性疑慮[编辑]
不少國外3C部落客都警告，Windows 10作業系統的隱私設定預設值，對於使用者的隱私相當具侵略性，且變更設定的步驟過於繁複，不易操作。 BGR News的記者就表示，在所有Windows 10的功能預設值中，會入侵使用者隱私的設定都被預設成同意，「只要登入Microsoft電子郵件，微軟就能讀取你的信件、聯絡人，甚至是行事曆等資料。」
一名網頁工程師波塔（Jonathan Porta）則表示，「我很驚訝Windows 10的預設值都不是很尊重使用者的隱私；我也相信，通常多數人會為了使用上的方便快速，直接同意這些預設值，而不曉得自己洩漏了多少個人資料及隱私給微軟。」
透過預設值的設定，微軟有正當理由可以取得用戶的聯絡人資料、行事曆資訊及其他相關輸入資料，作業系統也希望能取得用戶的所在位置及過去的所在位置，而這些資料不僅提供給微軟，甚至是微軟的合作夥伴[51]。

升级和支持[编辑]
Windows 10接受服务的方式与之前的版本大相径庭，它會強制自動更新。Terry Myerson解释道：「微软的目的将淡化人们常论的『你用的Windows是哪个版本』，因此将以做为服务性的软件形式发放。」[67][68][69]
不同于前些版本，Windows Update已经不允许有选择性的安装更新，而是所有更新一同強制性自动下载并安装，这其中包含补丁、功能性更新以及驱动程序。用户的選擇，只有在系统闲置时自动安装更新或通知以安排重新启动的時間點。[70][71]Windows Update也允许端对端（P2P）系统分发更新，這會使用戶的頻寬被佔去，因此微軟讓這個功能可以關閉。默认情况下用户的带宽将配合微软服务器分发更新至其它用户电脑中；在局域网环境中用户可以单独使用端对端更新他们的电脑。[72]
专业版和企业版允许用户选择商用当前分支（Current Branch for Business）释放渠道（在Windows Update设置中被称为「延迟升级」），功能性升级的接收将大约延迟到主流释放的四个月后。 [73][71]商用当前分支可以将功能性升级最大错后八个月，此后更新必须安装，以便维护安全性更新的支持和索取。管理员也可使用“Windows Update for Business”系统来组织整个网络的更新波次。[71]企业版还可使用长期支持分支（Long-term support branch，LTSB）。[73]Windows 10的LTSB里程碑是商用当前分支周期性的快照，并且将在超过十年的支持周期内仅接收关键性补丁。系统也可被置于错后最新LTSB版本的一到两个版本，以允许结构性的部署和内部生命周期 。[67][74][75]微软的主管Stella Chernyak解释道：“我们有很多需要关键性环境的企业，他们想要很长一段时间测试并稳定化这个环境，而我们尊重这个事实。”[76]
暫停更新的套件[编辑]
微軟接受到許多對於強制更新的抱怨，所以提供官方的套件予使用者，給他們可以用來暫停更新。[77]
對DDE的支援[编辑]
参见：動態資料交換
Windows 10不支援DDE（動態資料交換），即便是將UAC調至最低等級。
最後一版Windows[编辑]
在微軟Ignite開發者大會上，該公司高層主管傑瑞·尼克遜透露，Windows 10將是這個Windows作業系統品牌的最後一版產品[78]。微軟發言人透露，對於Windows作業系統的發展方向，微軟已經有新的計劃，Windows作業系統將在未來改變透過盒裝軟體的一次性售賣模式，調整為線上軟體服務，為用戶提供經常性的升級和更新；至於是否會沿用Windows品牌，目前仍無定論。
Windows將不再有新版本問世，取而代之的將是定期的改進和更新。得以採用這種方式的部分原因是，把开始功能表和內建應用程式等作業系統部件分解為獨立部分，各個部件可以獨立更新，組成完整的Windows核心作業系統。[79][80][81]
另外，微軟主管表示，微軟考慮開放Windows作業系統的原始碼，跟進Linux、Android等廣受歡迎的開放原始碼作業系統。
開放Windows系統的原始碼，可望幫助微軟進一步提高Windows使用率，吸引更多人使用微軟自家的應用程式。開放程式碼供外部技術人員檢視，還能更容易抓出公司工程師沒找到的程式漏洞。微軟開放作業系統的原始碼後，也能仿效Linux和Android的獲利模式，靠著幫大企業包裝、銷售和更新作業系統賺取獲利。[82]
